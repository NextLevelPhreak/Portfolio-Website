Website Style Guide

Concept:

This is a style guide for one of the region's largest e-commerce businesses. Their employees kept forgetting their brand colours and using shades of grey, as if entranced by weird skeleton magic. This style guide was built so that the employees can keep everything on-brand.

Step 1:

The first step was figuring out the logic behind the game, and creating pseudo-code to rough out the JavaScript code. I did this so that I could try some early designs without worrying about getting stuck trying to maintain an inconsistent interface. It was also useful to get an idea of what elements worked together, what worked out of place, and what could be changed to improve the game experience.

Step 2:

Next I worked on actually creating a model. I found the JavaScript APIs and started playing around with it to create some of the basic models. Some basic structures became extremely important: triangles for polygons.

So I create an index.html, with this file I create another file, js/app.js. Basically, I worked through a "Master" of the day project and built up more and more confidence with the language.

Core Technologies:

  • HTML
  • CSS
  • JavaScript

Final Result:

The final version of this project was just setting up SASS folders and having everything organized.
